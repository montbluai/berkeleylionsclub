# 🚀 QUICK START - Board Members

## How to Upload Photos or Events

### Step 1: Get to the Admin Dashboard
1. Go to **berkeleylionsclub.org**
2. Scroll to the **bottom** of the page
3. Click **"Admin Login"** (bottom right corner)

### Step 2: Login
- Password: `berkeley2025`
- Click "Login to Admin Dashboard"

### Step 3: Choose What You Want to Do

#### 📸 To Upload Photos:
1. Make sure **"Photo Gallery"** tab is selected (it's the default)
2. Drag & drop your photo OR click to select it
3. Add a caption describing the photo
4. Choose a category from the dropdown
5. Click **"Upload Photo"**
6. Done! Photo appears on the Photo Gallery page immediately

#### 📅 To Add a Volunteer Event:
1. Click the **"Featured Events"** tab at the top
2. Fill out the form:
   - Upload an event poster image
   - Enter event name (e.g., "2026 Crab Feed")
   - Write a description
   - Set date and time
   - Add location details
   - Specify how many volunteers needed
   - List volunteer tasks (one per line)
   - Set ticket price or mark as "Free"
   - (Optional) Add Square payment link
   - (Optional) Check "To-go available" if applicable
3. Click **"Publish Featured Event"**
4. Done! The next upcoming event shows on the Volunteer page

### Step 4: Managing Existing Content

#### Delete a Photo:
1. Go to **Photo Gallery** tab
2. Hover over any photo
3. Click the **red trash can** icon
4. Confirm deletion

#### Delete an Event:
1. Go to **Featured Events** tab
2. Scroll to "Existing Events" section
3. Click the **trash can** icon next to the event
4. Confirm deletion

### Step 5: Logout
- Click **"Logout"** in the top right corner
- You can close the browser anytime

## 💡 Pro Tips

### For Photos:
- **Categories**: Service, Fundraising, Community, Events, Fellowship
- **Max size**: 10MB per photo
- Photos appear immediately on the gallery

### For Events:
- **Past events are automatically hidden** from the public Volunteer page
- Only the **next upcoming event** displays prominently
- You don't need to delete old events - they're marked as "Past Event"
- **Volunteer tasks**: Enter each task on a new line
- **Pricing**: Check "Free" OR enter a price like "$80"
- **Square links**: Add your Square payment link for easy ticket purchases
